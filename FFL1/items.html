ITEMS & GEAR
ROM-EXTRACTED DATA ONLY
Data source notice: Item names, type classifications, use counts, and GP prices are extracted directly from ROM (0x14640 name table, 0x17E10 price table — 6-digit BCD format). Numeric values such as base power, elemental affinity, and stat bonuses have not yet been extracted from the ROM and are omitted.

Weapons are classified by type byte: 0x32 = sword/blade (prefix 0xEC renders as ' in-game), 0x14/0x1E = other weapon (gun/ranged).

NAMEROM TYPEUSESGP

Armor entries have type byte 0xFE or 0xFD. Slot (helmet/chest/gloves/shoes) is determined by prefix byte. Stat bonuses are not in the name record and have not been extracted.

NAMESLOT (prefix byte)GP

Spells and abilities have prefix byte 0xEF (elemental spell) or type bytes 0x05, 0x0A, 0x0B, 0x0F (ability). Power, target, element, and status effects have not been extracted from the ROM.

NAMEROM TYPEUSESGP

Usable items have type byte 0x01 or 0x03. Effects (heal amount, stat boost, status cure) have not been extracted from the ROM.
