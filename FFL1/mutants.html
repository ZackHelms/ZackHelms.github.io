Data source notice: Mutant stat-growth rates, ability-gain probabilities, and stat cap values have not been extracted from the ROM binary. Only the ability database (names and type classifications from the ROM) is presented here.

ABILITY DATABASE (ROM DATA)

All 252 entries from the ability table at 0x14640. Each entry is 8 bytes: names use the FFL character encoding (0x8A='A'…0xA3='Z', 0x80–0x89='0'–'9'), type from byte 7, prefix from byte 0, uses count from byte 7 of the name record, GP from 0x17E10. Filter by name or type below.

Type classification is based on the type byte (byte 7) and prefix byte (byte 0) of each ROM entry. See the General page for the full type-code reference table.
