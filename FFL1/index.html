QUICK REFERENCE

THREE CLASSES

Human — Equipment-based stats. 8 item slots. Grows via stat-boosting items.

Mutant — Random stat gains after battle. 4 item + 4 ability slots. High magic potential.

Monster — No equipment. Transforms by eating meat. 4 meat slots. Most powerful end-game.

STAT CAPS

All stats display a maximum of 99 on screen.

Internal maximum is 255 for STR/DEF/AGI/MANA.

HP maximum is 999.

Mutant MANA can grow beyond 99 (displayed as 99 but keeps scaling spells).

FIVE WORLDS

World 1 → World 2 → World 3 → World 4 → Tower of God

Collect 4 Crystal Balls per world to open the gate to the next.

The Tower of God has 20 floors leading to the final bosses: Ashura and Creator.

ABOUT THIS PORT

This is a from-scratch JavaScript recreation of FFL1 built on ROM-extracted game data. Monster names, HP values, stats, and ability names were extracted directly from the original ROM binary. Game mechanics and world structure are reconstructed from community documentation.

All game data lives in data/*.json — edit those files to modify monster stats, weapons, shops, encounter tables, maps, or story text. The JS engine reads the data at runtime so you never need to touch the engine code for content changes.
