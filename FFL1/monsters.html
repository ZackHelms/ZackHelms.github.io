MONSTER CLASS
ROM-EXTRACTED DATA ONLY
Data source notice: Stats are extracted directly from the ROM. Mechanics governing how the Monster class plays (transformation rules, meat slot count, trigger conditions) have not been extracted from the ROM binary and are not presented here.

MEAT DROP DATA (ROM)

Each monster entry includes a drop byte at position 6 of the stat record at 0x1AAE8. From this byte we extract:

  • can_drop_meat — whether this monster can drop a meat item
  • meat_level — the level encoded in the drop byte

Note: what the meat_level value controls in gameplay has not been verified from ROM code. The meat system mechanics (eat count to transform, trigger conditions, transformation target) are not yet extracted.

ALL MONSTERS (ROM DATA)

All 200 entries from the ROM monster table. HP from 0x1B254 (2-byte LE). STR/DEF/AGI/MANA from 0x1AAE8 bytes 2–5 per entry. Values exceeding 99 are highlighted — 99 is a display cap, not the stored value.
