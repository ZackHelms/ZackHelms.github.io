ROM-EXTRACTED DATA ONLY
Data source notice: Monster stats and the meat transformation table are extracted directly from the ROM. Trigger conditions (how many meats eaten, combat conditions) and meat slot count have not been extracted from the ROM binary. The transformation classes section below shows what you can become, not when.

ALL MONSTERS (ROM DATA)

All 200 entries from the ROM monster table. HP from 0x1B254 (2-byte LE). STR/DEF/AGI/MANA from 0x1AAE8 bytes 2–5 (9-byte stride). Gold from BCD table at 0x1B2A4 via lower nibble of stat byte 6. Values exceeding 99 are highlighted — 99 is a display cap in-game, not the stored value. Ability lists from pointer at stat bytes 7–8 (bank-6 CPU address), decoded from item stat table at 0x1B700. Cyan = attack, dim = passive defense (D/E items).

MONSTER CLASSES & MEAT TRANSFORMATION

There are 25 monster classes (IDs 0–149; class = ID ÷ 6). When a monster eats meat, it picks a random result class (roughly equal chance for each of the 25 classes), then picks a random member of that class weighted by the values below. Gold names = highest probability in that class. Monsters 150+ (human warriors, bosses) cannot eat meat.
