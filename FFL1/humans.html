HUMAN CLASS
ROM-EXTRACTED DATA ONLY
Data source notice: Human class mechanics — equipment slot counts, stat item amounts, stat caps, and armor bonus values — have not been extracted from the ROM binary. Only the armor name and slot-type data from the ROM ability table is shown below.

ARMOR ENTRIES (ROM DATA)

Armor items are identified in the ability table at 0x14640 by type byte 0xFE or 0xFD. Slot type (helmet, chest, gloves, shoes) is determined by the prefix byte at position 0. Names are decoded using the FFL character encoding. Stat bonuses are not encoded in the ability name record and have not been extracted.

STAT ITEM ENTRIES (ROM DATA)

Items that affect character stats are classified as usable_item type (type byte 0x01 or 0x03) in the ability table. The names below are ROM-decoded. What each item does numerically has not been extracted from the ROM code.
